# twitter-media-scraper
用于提取推特页面中包含的媒体数据（例如图片）的脚本

支持输入如下两种格式的链接：

1. https://<span></span>twitter.com/\*\*\*/status/\*\*\*\*\*\*

2. https://<span></span>t.co/******

获取到的媒体文件自动保存到路径下的download文件夹

默认使用系统代理，无需配置，若提取出现任何问题，请反馈log文件
